# Invitación Baby Shower - Matías 👶

Invitación digital interactiva y animada para el Baby Shower de Matías.

## 🎉 Características

- ✨ **Pantalla de apertura animada** con globos flotantes
- 🎵 **Música de fondo** que se reproduce automáticamente
- ⏳ **Contador regresivo** dinámico en tiempo real (Días : Horas : Minutos : Segundos)
- 📱 **Completamente responsive** - Funciona en todos los dispositivos
- 🗺️ **Ubicación en Google Maps** - Botón para ver la ubicación exacta
- 💬 **Confirmación por WhatsApp** - Botones para contactar papá o mamá
- 🖼️ **Imágenes familiares** - Fotos de papás y bebé
- 🎨 **Diseño elegante** - Colores celeste, blanco y dorado
