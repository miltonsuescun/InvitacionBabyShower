# 🎉 Invitación Baby Shower - Matías

Invitación digital interactiva y animada para el Baby Shower de Matías, construida con HTML5, CSS3 y JavaScript vanilla.

## ✨ Características

- 🎬 **Secuencia animada de 3 fases** — Loader → Apertura → Invitación
- 🐻 **Ositos CSS** — Mascota dibujada completamente con divs (sin imágenes)
- 🎈 **Globos flotantes** — Animaciones continuas en toda la página
- 🎵 **Música de fondo** — Reproducción automática al llegar a la invitación
- ⏳ **Countdown en vivo** — Días, horas, minutos y segundos en tiempo real
- 📷 **Modal de fotos clickeables** — Abre a pantalla completa (papás y bebé)
- 📱 **Completamente responsive** — Mobile-first, funciona en todos los dispositivos
- 🗺️ **Google Maps integrado** — Botón con ubicación exacta del evento
- 💬 **Confirmación WhatsApp** — Botones para contactar papá y mamá
- 🔗 **Link preview optimizado** — Foto en redes sociales y WhatsApp

## 📝 Notas de Desarrollo

- Las animaciones se pueden personalizar modificando `@keyframes` en `style.css`
- El countdown se basa en la fecha del navegador del usuario
- La música se inicia automáticamente con autoplay (con manejo de restricciones de navegador)
- El modal de fotos se cierra con ESC, click externo o botón X
